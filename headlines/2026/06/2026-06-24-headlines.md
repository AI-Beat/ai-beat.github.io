# AI Headlines — 2026-06-24

- [Qwen-AgentWorld: Language World Models for General Agents](https://arxiv.org/abs/2606.24597) — Alibaba/Qwen releases two open-weight language world models (35B-A3B and 397B-A17B) trained on 10M+ environment-interaction trajectories to simulate seven digital-agent domains (MCP, Search, Terminal, SWE, Android, Web, OS) via long chain-of-thought reasoning; models serve both as offline RL training environments and as warm-start foundations for downstream agent tasks. *(June 23, 2026)*

- [FUTO Swipe – A new swipe typing model](https://swipe.futo.tech/) — FUTO releases open models powering swipe gesture typing in FUTO Keyboard: a three-model stack (Encoder 635K, ContextLM 1.5M, Decoder 304K params — 2.5M total) that achieves 26% fewer errors than Gboard on their benchmark; trains on one workstation GPU; first freely licensed and open swipe model. *(June 13, 2026)*

- [Mistral OCR 4](https://mistral.ai/news/ocr-4/) — Structured document extraction with per-block bounding boxes, type classification (title, table, equation, etc.), and per-word confidence scores across 170 languages; tops OlmOCRBench at 85.20; self-hostable for enterprise; API priced at $4/1K pages. *(June 23, 2026)*

- [lift: 9B vision model for structured JSON from documents](https://github.com/datalab-to/lift) — Datalab releases an open-weights 9B VLM that takes a JSON schema and returns matching structured output from PDFs/images; schema-constrained decoding guarantees valid JSON; trained abstention returns null rather than hallucinating absent fields; 90.2% field accuracy on 225-document benchmark. *(June 23, 2026)*
