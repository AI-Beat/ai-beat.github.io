# AI Headlines — 2026-07-09

- [OpenAI Finds ~30% of SWE-bench Pro Tasks Are Broken](https://openai.com/index/separating-signal-from-noise-coding-evaluations/) — OpenAI's audit of the 731-task SWE-bench Pro public split found 27–34% of tasks are broken due to hidden requirements, contradictory instructions, overly strict tests, or incomplete grading criteria; model developers are advised to manually verify both failures and successes. *(July 8, 2026)*

- [Databricks Benchmarks Coding Agents on Its Own Multi-Million-Line Codebase](https://www.databricks.com/blog/benchmarking-coding-agents-databricks-multi-million-line-codebase) — Databricks built an internal eval from real merged PRs in their polyglot codebase (Python, Go, TypeScript, Scala, etc.), validating correctness via test execution rather than LLM judges; GLM 5.2 tied with Opus 4.8 on quality at roughly two-thirds the per-task cost, and harness choice caused 2x+ cost swings across identical models. *(July 8, 2026)*

- [Microsoft Releases Flint, a Visualization Language for AI Agents](https://microsoft.github.io/flint-chart/) — Flint is a compact, semantically typed DSL that compiles into Vega-Lite, Apache ECharts, or Chart.js, giving AI agents a shorter, cleaner spec target for chart generation instead of raw Vega-Lite JSON; ships with an MCP server for agent integration. *(July 8, 2026)*
