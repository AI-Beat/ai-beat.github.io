# AI Headlines — 2026-07-17

- [Kimi K3: Open Frontier Intelligence](https://www.kimi.com/blog/kimi-k3) — Moonshot AI announces Kimi K3, a 2.8T total / ~50B active MoE model claiming "the world's first open 3T-class model"; benchmarks show strong coding and reasoning performance, with weights promised by July 27. *(July 16, 2026)*

- [Ring-Zero: Scaling Zero RL to a Trillion Parameters for Emergent Reasoning](https://arxiv.org/abs/2607.12395) — Ring-2.5-1T-Zero scales verifiable-reward-only RL to 1T parameters; emergent behaviors include self-verification, parallel reasoning, and "context anxiety," following a consistent discovery-then-sharpening training dynamic. *(July 14–16, 2026)*

- [LM Studio Bionic: the AI agent for open models](https://lmstudio.ai/blog/introducing-lm-studio-bionic) — LM Studio releases a separate agentic product for open models with local voice transcription via Voxtral (Mistral), inline code diffs, document processing, and a privacy-first cloud option; positioned against Cursor and Windsurf for the open-model crowd. *(July 16, 2026)*

- [NotebookLM is now Gemini Notebook](https://blog.google/innovation-and-ai/products/gemini-notebook/notebooklm-gemini-notebook/) — Google rebrands its popular research and note-taking product; the functionality remains largely unchanged but the new name anchors it firmly in the Gemini product line. *(July 17, 2026)*
