# AI Headlines — 2026-07-03

- [Is One Layer Enough? Training A Single Transformer Layer Can Match Full-Parameter RL Training](https://arxiv.org/abs/2607.01232) — A Qwen-family study finds RL gains concentrate in a narrow band of middle transformer layers (≈40–60% depth); training the single best layer can surpass full-parameter training, while layers near the input and output contribute little; introduces "layer contribution" as a metric and shows layer rankings remain stable across models, algorithms, and task domains. *(July 1, 2026)*

- [Miles: A PyTorch-Native Stack for Large-Scale LLM RL Post-Training](https://pytorch.org/blog/miles-a-pytorch-native-stack-for-large-scale-llm-rl-post-training/) — RadixArk's open-source framework composes SGLang (rollout), Megatron-LM (training), and Ray (orchestration) into a unified RL pipeline with disaggregated/colocated execution, async rollout-training overlap, MoE-aware Routing Replay to preserve expert assignments across the boundary, and full INT4-QAT support for 1TB-scale models on single-node VRAM. *(June 30, 2026)*

- [claude-real-video: Frame-Sampling Pipeline for LLM Video Understanding](https://github.com/HUANGCHIHHUNGLeo/claude-real-video) — Open-source tool that lets any LLM "watch" a video locally: detects scene changes for adaptive frame sampling, deduplicates near-identical frames with pixel-difference sliding windows, transcribes audio via Whisper or embedded subtitles, and produces a single manifest consumable by any vision-capable model without needing a native video API. *(Surfaced July 3, 2026)*
