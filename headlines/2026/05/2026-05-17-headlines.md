- [SANA-WM: Efficient Minute-Scale World Modeling with Hybrid Linear Diffusion Transformer](https://arxiv.org/abs/2605.15178) — NVIDIA's open-source 2.6B-parameter world model generates 60-second 720p video with 6-DoF camera control on a single GPU; distilled variant completes a full clip in 34 seconds on an RTX 5090, achieving 36× higher throughput than prior open-source baselines. *(May 14, 2026)*

- [δ-mem: Efficient Online Memory for Large Language Models](https://arxiv.org/abs/2605.12357) — A compact 8×8 associative memory state updated by delta-rule learning augments a frozen full-attention backbone at inference time, applying low-rank corrections to attention with no fine-tuning or backbone replacement; achieves 1.31× gains on MemoryAgentBench. *(May 12, 2026)*

- [Zerostack – A Unix-inspired coding agent written in pure Rust](https://github.com/gi-dellav/zerostack) — Minimal coding agent (~7k LoC) with multi-provider support and an idle RAM footprint of ~8MB; v1.0.2 released today with MCP support, iterative loop system, and git worktree integration. *(May 17, 2026)*
