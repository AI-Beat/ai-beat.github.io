# AI Headlines — 2026-05-06

- [Agents can now create Cloudflare accounts, buy domains, and deploy](https://blog.cloudflare.com/agents-stripe-projects/) — Cloudflare and Stripe released a protocol during Agents Week that lets AI agents autonomously provision accounts, register domains, start paid subscriptions, and deploy applications using Stripe for identity attestation and tokenized payments; default spending cap is $100/month per provider with no raw card details exposed to the agent. *(April 30, 2026)*

- [Accelerating Gemma 4: faster inference with multi-token prediction drafters](https://blog.google/innovation-and-ai/technology/developers-tools/multi-token-prediction-gemma-4/) — Google releases open Apache 2.0 MTP draft models for the full Gemma 4 family; the drafters share the target model's embedding table and last-layer activations to predict several tokens ahead, which the target model verifies in parallel, achieving up to 3x throughput gains on dense models; MoE variants like Gemma 4 26B face harder tradeoffs due to expert routing variance. *(May 5, 2026)*

- [Google Chrome silently installs a 4 GB AI model on your device without consent](https://www.thatprivacyguy.com/blog/chrome-silent-nano-install/) — On a fresh macOS profile with zero user interaction, Gemini Nano's 4 GB `weights.bin` appeared within 15 minutes via Chrome's component-updater; the file auto-redownloads after deletion, and the author estimates the silent push to ~500 million active Chrome users represents roughly 30,000 tonnes of CO2-equivalent from network transfer alone. *(May 4, 2026)*
