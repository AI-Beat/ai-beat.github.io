# AI Headlines — 2026-05-04

- [Are Tools All We Need? Unveiling the Tool-Use Tax in LLM Agents](https://arxiv.org/abs/2605.00136) — New arxiv paper introduces the concept of a "tool-use tax": in controlled experiments isolating prompt formatting, protocol overhead, and actual execution gains, tool-augmented reasoning doesn't outperform native chain-of-thought when semantic distractors are present; authors propose G-STEP, a lightweight inference-time gating mechanism to decide when tool calls are warranted. *(May 3, 2026)*

- [Position: Agentic AI Orchestration Should Be Bayes-Consistent](https://arxiv.org/abs/2605.00742) — Thirty-author ICML 2026 position paper argues that the orchestration layer of LLM agents should apply Bayesian decision theory—maintaining calibrated probability distributions over task-relevant unknowns and selecting actions (including tool calls) by expected utility—rather than the ad hoc heuristics most frameworks use today. *(May 1, 2026)*

- [Agentic Coding Is a Trap](https://larsfaye.com/articles/agentic-coding-is-a-trap) — Lars Faye's piece, published April 30 and picking up 313 upvotes on Hacker News, argues that heavy reliance on AI coding agents produces skill atrophy in developers: the same critical thinking that heavy AI usage erodes is exactly what's needed to supervise AI outputs, and vendors have a financial incentive to obscure this tradeoff. *(April 30, 2026)*

- [Cisco Releases Open Source Tool for AI Model Provenance](https://www.securityweek.com/cisco-releases-open-source-tool-for-ai-model-provenance/) — Cisco's Model Provenance Kit (Python CLI + Hugging Face fingerprint dataset) generates model identity signals from metadata, tokenizer similarity, and weight-level probes; operates in compare mode (shared lineage between two models?) and scan mode (closest match in a reference database?), targeting supply chain verification and license compliance for open-weight deployments. *(May 1, 2026)*

- [DeepClaude: Claude Code Agent Loop Running on DeepSeek V4 Pro](https://github.com/aattaran/deepclaude) — Shell-script wrapper and local proxy that intercepts Claude Code's API calls and routes them to cheaper backends (DeepSeek V4 Pro at $0.87/M output tokens vs Claude Opus at $15/M), keeping Claude's UX and agent loop intact while substituting the underlying model; hit 349 points on Hacker News. *(May 4, 2026)*
