# AI Headlines — 2026-05-01

- [Granite 4.1: IBM's 8B Dense Model Matches the Previous 32B MoE](https://research.ibm.com/blog/granite-4-1-ai-foundation-models) — IBM releases Granite 4.1 (3B/8B/30B, Apache 2.0) with a dense transformer architecture; the 8B instruct variant matches or beats the prior Granite 4.0 32B MoE model on ArenaHard, tool-calling, and math benchmarks, after a five-phase training pipeline that includes LLM-as-Judge fine-tuning filtering and a four-stage RL curriculum with a dedicated math-recovery phase. *(April 29, 2026)*

- [Shai-Hulud Themed Malware Found in the PyTorch Lightning Library](https://semgrep.dev/blog/2026/malicious-dependency-in-pytorch-lightning-used-for-ai-training/) — PyPI package `lightning` versions 2.6.2 and 2.6.3 were compromised with credential-stealing malware that, on import, launches a background thread, downloads the Bun JavaScript runtime, and executes an 11 MB obfuscated payload harvesting SSH keys, cloud credentials, Git tokens, and crypto wallets; detected 18 minutes after upload, quarantined by PyPI. Part of the ongoing Mini Shai-Hulud campaign that previously hit LiteLLM, Telnyx, and Xinference. *(April 30, 2026)*

- [Less Is More: Cognitive Load and the Single-Prompt Ceiling in LLM Math Reasoning](https://arxiv.org/abs/2604.18897) — Paper tests 40+ prompt variants on equational theory problems and finds performance saturates at ~60–79% despite a +19.5pp gain over baseline; weaker models (Llama 3.3 70B) collapse to 0% TRUE recall when prompts exceed 2KB, suggesting capacity constraints create a hard ceiling that better prompt engineering cannot overcome. *(April 30, 2026)*
