# AI Headlines — 2026-05-16

- [Orthrus: Lossless Parallel Decoding via Dual-View Diffusion](https://arxiv.org/abs/2605.12825) — A trainable diffusion head is grafted onto a frozen AR backbone, sharing the same KV cache; an exact consensus mechanism validates all proposed tokens through the AR path, guaranteeing output-distribution fidelity while achieving up to 7.8× speedup (5.36× average) on Qwen3-8B with O(1) memory overhead and only 16% additional trainable parameters. *(May 12, 2026)*

- [Mistletoe: Stealthy Acceleration-Collapse Attacks on Speculative Decoding](https://arxiv.org/abs/2605.14005) — A new adversarial attack crafts inputs that preserve the target model's visible output quality while causing the draft model to disagree, collapsing acceptance length and throughput without leaving detectable accuracy fingerprints; uses null-space projection to decouple the degradation gradient from the semantic-preservation direction. *(May 13, 2026)*

- [Self-Distilled Agentic Reinforcement Learning (SDAR)](https://arxiv.org/abs/2605.15155) — A sigmoid-gated self-distillation auxiliary objective layered on top of GRPO for multi-turn agent RL; improves over the GRPO baseline by +9.4% on ALFWorld, +10.2% on WebShop, and +7.0% on Search-QA across Qwen2.5 and Qwen3 model families. *(May 14, 2026)*
