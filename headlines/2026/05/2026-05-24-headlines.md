# AI Headlines — 2026-05-24

- [MOSS: Self-Evolution through Source-Level Rewriting in Autonomous Agent Systems](https://arxiv.org/abs/2605.22794) — Autonomous agents that can rewrite their own source code in response to production failures, not just text artifacts like prompts and skill files; tested on OpenClaw where a single evolution cycle improved a four-task benchmark from 0.25 to 0.61 by fixing an unreachable harness-level bug. *(May 21, 2026)*

- [DelTA: Discriminative Token Credit Assignment for Reinforcement Learning from Verifiable Rewards](https://arxiv.org/abs/2605.21467) — Identifies that RLVR gradient signals are dominated by shared high-frequency formatting tokens rather than tokens that actually distinguish good from bad responses; discriminator-based reweighting yields +3.26/+2.62 avg points on seven math benchmarks with Qwen3-8B/14B-Base over DAPO baseline. *(May 20, 2026)*

- [xAI launches Grok Build 0.1 for agentic coding with 256k context](https://docs.x.ai/developers/models/grok-build-0.1) — Purpose-built coding model designed to plan, write code, use tools, and self-correct in a continuous loop; $1/$2 per million input/output tokens, available to SuperGrok and X Premium subscribers via OpenCode. *(May 21, 2026)*
