- [CrabTrap: An LLM-as-a-judge HTTP proxy to secure agents in production](https://github.com/brexhq/CrabTrap) — Brex open-sources a Go MITM proxy that intercepts all outbound HTTP from AI agents and evaluates each request against a natural-language security policy; uses a two-tier engine (static URL rules first, LLM judge for ambiguous cases) with full PostgreSQL audit logging. *(April 17, 2026)*
