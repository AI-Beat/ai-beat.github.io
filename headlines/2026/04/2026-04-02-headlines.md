# AI Headlines — 2026-04-02

- [Trinity Large Thinking: Arcee AI's open 400B reasoning model](https://www.arcee.ai/blog/trinity-large-thinking) — Arcee AI releases the Thinking variant of their 400B sparse MoE model; #2 on PinchBench for agentic tasks at $0.90/M output tokens, roughly 96% cheaper than the #1 model (Claude Opus 4.6); Apache 2.0; 512K context; 256 experts with 4 active per token (~13B params per forward pass). *(April 1, 2026)*

- [SALOMI: extreme low-bit transformer quantization research](https://github.com/OrionsLock/SALOMI) — Research repo examining whether post-hoc 1-bit quantization can match ternary approaches; central finding is that "strict 1.00 bpp post-hoc binary quantization does not hold up under rigorous evaluation" for GPT-2-class models; practical results cluster around 1.2–1.35 bpp using Hessian-guided vector quantization or magnitude-recovery techniques. *(April 2, 2026)*

- [The Revenge of the Data Scientist](https://hamel.dev/blog/posts/revenge/) — Essay arguing that AI hasn't automated away data scientists because most of the job was never model training — it was building evaluation harnesses, designing metrics, and debugging stochastic systems; identifies five failure modes in AI evaluation: generic metrics, unvalidated LLM judges, synthetic test sets, poor labeling, and over-automation; core prescription: "always look at the data." *(April 1, 2026)*

- [CC Gateway: reverse proxy to strip Claude Code telemetry](https://www.emsi.me/tech/ai-ml/surveillance-showdown-claude-codes-data-harvesting-sparks-privacy-rebellion-with-cc-gateway/2026-04-01/183a00) — Community tool sitting between Claude Code and the Anthropic API, normalizing device fingerprint data and stripping 640+ telemetry event types reported to fire every 5 seconds; reduces the 40+ machine fingerprint dimensions sent to Anthropic without breaking functionality. *(April 1, 2026)*

- [Meta BOxCrete: Bayesian optimization for concrete mix design](https://engineering.fb.com/2026/03/30/data-center-engineering/ai-for-american-produced-cement-and-concrete/) — Meta open-sources BOxCrete (built on their Adaptive Experimentation platform), which learns from historical mix designs and refines predictions with each new lab test; deployed at a Minnesota data center, the AI-optimized mix achieved 43% faster full structural strength and ~10% reduction in cracking risk using domestically sourced materials. *(March 30, 2026)*

- [Together AI Aurora: RL-based adaptive speculative decoding](https://www.together.ai/blog/aurora) — Open-source framework that reframes speculative decoding as an RL problem: draft model is a policy, target model is the environment, accepted tokens are rewards; unlike static speculators that degrade as serving distributions shift, Aurora learns online from live inference traces and recovers within ~10,000 requests after a domain shift; 1.25× additional speedup over well-trained static speculators on Qwen3 and Llama3. *(March 31, 2026)*

- [Z.ai GLM-5V-Turbo: native multimodal vision coding model](https://www.marktechpost.com/2026/04/01/z-ai-launches-glm-5v-turbo-a-native-multimodal-vision-coding-model-optimized-for-openclaw-and-high-capacity-agentic-engineering-workflows-everywhere/) — Zhipu AI releases a model with genuinely native vision fusion (CogViT encoder + MTP architecture), 200K context; scored 94.8 on Design2Code vs. Claude Opus 4.6's 77.3; takes a UI wireframe and generates a full runnable frontend; optimized for OpenClaw agentic loops. *(April 1, 2026)*
